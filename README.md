 # Homework Helper Web App (Groq)

 A small homework helper web app that uses a Groq generative endpoint for General, Math, and Science questions.

 ## Setup

 1. Install Node.js 18 or newer.
 2. Open a terminal in `C:\Users\Donovan\homework-helper-webapp`.
 3. Run:
 	 ```powershell
 	 npm install
 	 ```
 4. Set your Groq API key:
 	 ```powershell
 ## Files

 - `index.html` — main web page
 - `style.css` — layout and styles
 - `script.js` — frontend logic that sends questions to the backend
 - `server.js` — Express backend that proxies requests to Groq
 - `package.json` — project dependencies and start script

 ## Notes

 - The server uses the official `groq-sdk` package and only requires `GROQ_API_KEY`.
 - If you get authorization or 403/404 errors, check your Groq project settings.
